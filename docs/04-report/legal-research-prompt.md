# 법률 리서치 자동화 프롬프트 (Claude Code + /chrome)

> **용도**: Claude Code에서 `/chrome` 브라우저 자동화를 활용한 3플랫폼 법률 리서치
> **작성일**: 2026-02-20
> **필수 환경**: Claude Code v2.0.73+, Chrome Extension v1.0.36+, 3플랫폼 로그인 상태

---

## 사전 조건

### 로그인 상태 확인
| 플랫폼 | URL | 계정 | 세션 특성 |
|--------|-----|------|----------|
| 엘박스 | lbox.kr | 서재홍 (스탠다드·풀 패키지) | 세션 유지 안정적 |
| 빅케이스 | bigcase.ai | iceamericano9 (Plus) | 세션 유지 안정적 |
| 슈퍼로이어 | superlawyer.co.kr | 서재홍의 워크스페이스 | 세션 만료 가능, 재로그인 필요할 수 있음 |

### Chrome 연결
```
1. Chrome 브라우저 실행 상태 확인
2. Claude Code에서 tabs_context_mcp 호출로 연결 확인
3. 각 플랫폼에 로그인된 상태인지 확인 (특히 슈퍼로이어)
```

## 프롬프트 1: 단일 플랫폼 검색

### 엘박스 판례 검색
```
/chrome 을 사용하여 엘박스(lbox.kr)에서 "{검색 키워드}"를 검색하고 결과를 정리해줘.

절차:
1. https://lbox.kr/v2 접속
2. 검색창에 키워드 입력 후 Enter
3. 검색 결과 건수와 상위 5건 판례 제목+사건번호 추출
4. 필요 시 AI 질의 탭으로 전환하여 AI 분석 결과도 추출
```

### 빅케이스 판례 검색 + AI 요약
```
/chrome 을 사용하여 빅케이스(bigcase.ai)에서 "{검색 키워드}"를 검색하고,
상위 판례의 AI 요약까지 추출해줘.

절차:
1. https://bigcase.ai/search/case 접속
2. 검색창에 키워드 입력 후 Enter
3. "검색 가이드" 팝업이 뜨면 "확인" 클릭
4. 전문판례 탭에서 결과 건수 확인
5. 상위 1건 클릭 → AI 요약 내용 추출
```

### 슈퍼로이어 AI 법률 상담
```
/chrome 을 사용하여 슈퍼로이어(superlawyer.co.kr)에서
"{법률 질문}"에 대해 AI 상담을 받고 결과를 정리해줘.

절차:
1. https://superlawyer.co.kr/chat 접속 (로그인 페이지 리다이렉트 시 알려줘)
2. 채팅 입력창에 질문 입력
3. "메시지 전송" 클릭
4. 약 50초 대기 후 답변 추출 (get_page_text)
5. 인용된 판례번호와 서적 출처를 별도로 정리
```

## 프롬프트 2: 3플랫폼 통합 리서치

```
/chrome 을 사용하여 "{법률 주제}"에 대해 3플랫폼 통합 리서치를 수행해줘.

### 작업 순서

**Step 1: 빅케이스 판례 검색**
- bigcase.ai에서 키워드 검색
- 검색 결과 건수 + 상위 3건 판례 (사건번호, 법원, 선고일, 주요 판시사항)
- 상위 1건 AI 요약 추출

**Step 2: 엘박스 AI 질의**
- lbox.kr AI 질의 탭에서 동일 주제 질문
- AI 답변 + 인용 판례 링크 + 참조 법조문 추출

**Step 3: 슈퍼로이어 AI 채팅**
- superlawyer.co.kr에서 동일 주제 질문
- AI 답변 + 인용 판례 + 서적 출처 추출

### 결과 병합 (IRAC 형식)

**Issue** (쟁점): {법률 주제}에서 핵심 법적 쟁점

**Rule** (법규):
- 관련 법조문 (3플랫폼 교차 확인)
- 핵심 판례 (중복 제거 후 정리)

**Application** (적용):
- 엘박스 AI 분석 요약
- 슈퍼로이어 AI 분석 요약
- 빅케이스 AI 요약

**Conclusion** (결론):
- 3플랫폼 공통 결론
- 플랫폼별 고유 인사이트
- 추가 검토 필요 사항
```

## 프롬프트 3: 문서 생성 연계

```
위 3플랫폼 리서치 결과를 바탕으로 {문서 유형}을 생성해줘.

### 문서 유형 선택:
- **법률 의견서**: IRAC 구조, 판례 인용 포함
- **노무자문 의견서**: 실무 관점, 조치사항 포함
- **근로계약서 검토 의견**: 조항별 적법성 분석
- **징계 검토 의견**: 정당성 요건 분석

### 생성 절차:
1. 리서치 결과에서 핵심 법리 추출
2. render_docx.py 활용하여 docx 생성 (또는 hwpx)
3. 판례 인용은 "대법원 YYYY. M. D. 선고 {사건번호} 판결" 형식
4. 서적 인용은 "저자, 『서명』, 출판사(연도), 페이지" 형식
```

## 자동화 팁

### 공통
- 각 플랫폼 탭은 별도 생성 (`tabs_create_mcp`)하여 병렬 작업 가능
- AI 답변 대기: `wait 30~60초` → `get_page_text`로 추출
- 연결 끊김 시: `tabs_context_mcp` 재호출로 복구

### 플랫폼별 주의사항
| 플랫폼 | 주의사항 |
|--------|---------|
| 엘박스 | AI 답변 중 "엘박스 AI가 답변 중입니다" 메시지 확인 |
| 빅케이스 | 첫 검색 시 "검색 가이드" 팝업 닫기 필요 |
| 슈퍼로이어 | 세션 만료 시 `/auth/login` 리다이렉트 → 수동 로그인 필요 |

### 셀렉터 요약
| 플랫폼 | 검색 입력 | 검색 실행 | AI 입력 | AI 전송 |
|--------|----------|----------|---------|---------|
| 엘박스 | `textbox` "키워드를 입력하세요" | Enter | textarea (AI탭) | 파란 화살표 버튼 |
| 빅케이스 | `textbox` "키워드를 검색하세요" | Enter | - | - |
| 슈퍼로이어 | - | - | `textbox` "요청하실 내용을 입력해 주세요" | `button "메시지 전송"` |

## 트러블슈팅

| 증상 | 원인 | 해결 |
|------|------|------|
| "Browser extension is not connected" | Chrome 확장 연결 끊김 | `tabs_context_mcp` 재호출 |
| 검색 결과 0건 | 검색어 미입력 상태로 카테고리 클릭 | 검색창 → 키워드 입력 → Enter 순서 준수 |
| 슈퍼로이어 로그인 페이지 | 세션 만료 | 수동 로그인 후 재시도 |
| AI 답변 미완료 | 대기 시간 부족 | wait 시간을 60초로 늘리기 |
| 빅케이스 팝업 차단 | 검색 가이드 모달 | "확인" 버튼 클릭 |
