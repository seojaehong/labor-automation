# 3플랫폼 통합 리서치 결과 (IRAC)

> **주제**: 부당해고 구제신청의 요건, 절차, 제척기간, 구제명령의 효력
> **실행일**: 2026-02-20
> **플랫폼**: 엘박스 + 빅케이스 + 슈퍼로이어

---

| 플랫폼 | 수집 방식 | 결과 |
|--------|---------|------|
| 엘박스 | 일반 검색 + AI 질의 | 12,072건 검색 + AI 답변 (5개 섹션, ~70초) |
| 빅케이스 | 판례 검색 + AI 요약 | 3,508건 검색 + 판례 상세 AI 요약 |

## Issue (쟁점)

부당해고를 당한 근로자가 노동위원회에 구제신청을 하는 경우:
1. 구제신청의 **요건** (신청권자, 신청대상, 해고의 개념)
2. 3개월 **제척기간**의 법적 성격과 기산점
3. 구제신청 **절차** (초심 → 재심 → 행정소송)
4. 구제명령의 **내용과 효력** (원직복직, 임금상당액, 이행강제금)

## Application (적용) — 플랫폼별 분석 비교

### 1. 엘박스 AI 분석 (가장 상세, 법리 중심)
- **강점**: 판례 인라인 링크 + 유권해석 + 실무서 참조까지 포함
- **구조**: 5개 섹션 (핵심 요약 → 법규범 → 판례법리 → 검토적용 → 한계고지)
- **고유 인사이트**:
  - 제척기간 기산점: 수령거부 시 도달인정(신의칙), 이메일 통지 유효 가능
  - 이행강제금: 임금상당액 지급 부분도 대상, 정년 도과 시 이행불능으로 무효 가능
  - 금전보상명령(제30조 제3항): 원직복직 대신 임금상당액 이상 금품 지급 가능
  - 금품청산(14일) 관계: 구제명령 금품은 제36조 14일 청산 대상 아님
- **한계**: 법조문 원문 미확보 인정, 대법원 판결 일부는 하급심 인용 형태

### 2. 슈퍼로이어 AI 분석 (체계적, 서적 인용 강점)
- **강점**: 법률 서적 4종 인용 (노동법 교과서급 출처)
- **구조**: 4개 섹션 (의의 → 요건 → 절차 7단계 → 판례 6건)
- **고유 인사이트**:
  - 권리구제 대리인 지원제도: 월 평균임금 300만원 미만 근로자 무료 법률 지원
  - 화해 제도: 화해조서 = 재판상 화해 동일 효력
  - 복직 거부와 불법행위: 신의칙상 배려의무 위반 법리
- **인용 서적**: 노동법의 쟁점과 사례(박영사), 인사노무 실무 핵심 노동법(박영사), 파산판례해설(박영사), 노동법사례연습(박영사)

### 3. 빅케이스 검색 (판례 수집 강점)
- **강점**: 3,508건 전문판례 + AI 요약 + AI 유사판례 + 쟁점별 판례보기
- **고유 인사이트**:
  - 대법원 1997. 2. 14. 선고 96누5926 AI 요약: "구제신청 기간은 제척기간, 행정심판법상 청구기간 규정 유추 적용 불가"
  - 상하급심 판례 추적 기능으로 법리 변천 확인 가능
- **활용**: 대량 판례 수집 후 필터링 (사건종류/법원/기간) → 특정 판례 AI 요약으로 빠른 법리 파악

## Conclusion (결론)

### 3플랫폼 공통 결론
1. **제척기간 3개월**: 도과 시 구제신청권 소멸, 정당한 사유 있어도 구제 불가, 행정심판법 유추 적용 불가
2. **구제명령의 성격**: 공법상 이행의무 부과 행정처분이며, 사법상 법률관계를 직접 변경하지는 않음
3. **절차**: 지방노동위원회(초심) → 중앙노동위원회(재심 10일) → 행정소송(15일) → 확정

### 플랫폼별 고유 인사이트 종합
| 주제 | 플랫폼 | 인사이트 |
|------|--------|---------|
| 제척기간 기산점 | 엘박스 | 이메일 통지도 유효, 수령거부 시 도달인정 |
| 무료 법률지원 | 슈퍼로이어 | 월 300만원 미만 근로자 → 공인노무사/변호사 무료 지원 |
| 금전보상명령 | 엘박스 | 원직복직 대신 임금상당액 이상 금품 지급 가능 |
| 화해 제도 | 슈퍼로이어 | 노동위원회 화해조서 = 재판상 화해 동일 효력 |
| 이행강제금 | 엘박스 | 정년도과 시 이행불능 → 이행강제금 무효 가능 |
| 복직 거부 | 슈퍼로이어 | 정당한 이유 없는 복직거부 → 불법행위 인정 가능 |

### 추가 검토 필요 사항
- 근로자성 판단 기준 (엘박스 AI가 한계로 명시)
- 구제명령의 민사집행 가능성 (집행권원성 쟁점)
- 이행강제금 상한액 최신 법령 확인
- 각 플랫폼 AI의 hallucination 가능성 → 판례 원문 교차검증 필수
